## 🔧 技术细节

### CSS技术
- **渐变背景**: `linear-gradient()`
- **渐变文字**: `-webkit-background-clip: text`
- **半透明**: `rgba()`
- **变换**: `transform: translateY()` / `translateX()`
- **阴影**: `box-shadow`
- **过渡**: `transition: all 0.3s ease`

### SCSS嵌套
```scss
.dashboard-container {
  .dashboard-header {
    .welcome-title { }
    .welcome-subtitle { }
  }
  .panel-group {
    .card-panel {
      &:hover { }
      .icon-people { }
    }
  }
}
```

### Flex布局
```scss
.card-header {
  display: flex;
  align-items: center;
  gap: 8px;
}

.action-buttons {
  display: flex;
  flex-direction: column;
  gap: 12px;
}
```
