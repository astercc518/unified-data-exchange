## 📝 项目特性

### 核心功能
- 🎨 Modern UI - Element UI 精美界面
- 👥 Multi-Role System - 多角色权限管理
- 📊 Data Management - 数据管理和处理
- 🛒 Order System - 订单管理系统
- 💰 Financial Management - 财务管理
- 🌐 Internationalization - 中英文国际化
- 📱 Responsive Design - 响应式设计
- 🔒 Security - JWT 认证和 RBAC

### 技术栈
- **前端**: Vue.js 2.6.10, Element UI 2.13.2
- **后端**: Node.js, Express 4.18.2, Sequelize 6.x
- **数据库**: MySQL
- **部署**: PM2, Nginx
